Privacy Policy

Effective date: 13 July 2026 · Version 1.0.0

RuTracker Visual Enhancer is a browser extension that improves the visual presentation, filtering, and navigation of supported tracker result pages. This policy explains the information processed by the extension and how it is used.

Summary

The extension does not send browsing data, page content, preferences, or usage information to its developer. It contains no analytics, advertising, tracking SDK, account system, or developer-operated server.

Information processed locally

To provide its visible features, the extension reads result rows and topic pages on supported tracker domains. This may include topic titles, URLs, poster and avatar URLs, short descriptions, author details, availability counts, dates, and original .torrent or Magnet links.

The extension stores the following information locally in the user's browser:

  • Interface preferences such as theme, language, poster size, visible columns, Visual view state, collapsed search state, and infinite-scroll preference.
  • A bounded cache of up to 300 topic records, automatically expiring after seven days.
  • Temporary navigation and scroll-restoration information for the current browser session.

Network requests

Topic details are requested directly from the same tracker origin that the user is viewing, using HTTPS and the browser's existing session. The extension may display external poster or avatar images embedded by those tracker pages. These requests go directly to the relevant website or image host, not through the extension developer.

Supported sites

Version 1.0.0 is configured exclusively for RuTracker.org and its subdomains. The extension only runs on the sites declared in its manifest.

Sharing and sale of data

The developer does not receive, sell, rent, share, or transfer user data. No data is used for advertising, profiling, creditworthiness, or lending. Humans do not review user browsing data.

Authentication

The extension does not read, copy, store, or transmit passwords, cookies, authentication tokens, or other credentials. Normal same-origin requests may include the browser session automatically, in the same way as navigation within the site.

External links

The options popup contains an optional Ko-fi support link. Ko-fi is contacted only after the user explicitly opens that link and its own privacy policy then applies. Original .torrent and Magnet links are opened only after a user action and are handled by the browser or the user's configured application.

Retention and deletion

Topic cache records expire after seven days and are limited to 300 entries. Preferences remain until changed, the extension's storage is cleared, or the extension is uninstalled. Session navigation data expires with the relevant browsing session. Users can delete all locally stored data by removing the extension or clearing its extension storage.

Chrome Web Store Limited Use

The use of information received from browser and website APIs is limited to providing and improving the extension's disclosed single purpose. Any use of information received from Google APIs will adhere to the Chrome Web Store User Data Policy, including its Limited Use requirements.

Children

The extension is not designed to collect information from children.

Changes

This policy may be updated when the extension's behavior changes. The effective date and version will be updated, and material changes to data practices will be disclosed before the changed practices begin.
