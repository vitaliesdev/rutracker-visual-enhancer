Help and troubleshooting

Support

Having trouble with the extension? Start with the quick checks below or contact the developer directly.

Email support Report an issue on GitHub

Before reporting a problem

  1. Confirm the extension is updated to version 1.0.0 or later.
  2. Reload the RuTracker page after changing the interface language.
  3. Temporarily disable other page-restyling or dark-mode extensions.
  4. Check whether the issue also appears with Visual view turned off.
  5. Include the affected page type, browser version, selected theme, interface language, and a screenshot when possible.

Please protect your account. Do not send passwords, cookies, private tracker credentials, Magnet links, or copyrighted files in a support request.

What support covers

Support covers the extension interface, its settings, and compatibility with RuTracker page layouts. The developer cannot provide tracker accounts, restore tracker access, troubleshoot torrent clients, or provide downloadable content.

Privacy questions

The extension has no analytics or developer-operated data collection. Read the complete privacy policy for details about local processing, storage, and retention.

Support development

If the extension saves you time and you want to help maintain it, Ko-fi contributions are optional and greatly appreciated.
