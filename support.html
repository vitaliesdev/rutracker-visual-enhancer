Support

For help with RuTracker Visual Enhancer, contact REPLACE_WITH_CONTACT_EMAIL or use the Chrome Web Store Support hub.

Before reporting a problem

  1. Confirm the extension is updated to version 1.0.0 or later.
  2. Reload the tracker page after changing the interface language.
  3. Temporarily disable other page-restyling or dark-mode extensions.
  4. Include the affected page type, browser version, selected theme, and interface language.
  5. Do not send passwords, cookies, private tracker credentials, Magnet links, or copyrighted files.

Scope of support

Support covers the extension interface and compatibility. The developer cannot provide tracker accounts, restore tracker access, troubleshoot torrent clients, or provide downloadable content.
